# 需求文档模板

```markdown
# [项目名] 需求文档

> 创建日期：[YYYY-MM-DD]
> 状态：Draft | Review | Approved

## 背景与目标

[一句话描述问题和解决方案]

## 范围

### 包含
- [ ] 功能 A
- [ ] 功能 B

### 不包含
- 明确不做的事情

## 约束

- **技术约束**：[框架、语言、依赖]
- **时间约束**：[截止日期]
- **资源约束**：[人员、预算]

## 验收标准

- [ ] 标准 1
- [ ] 标准 2

## 风险

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| ... | 高/中/低 | 高/中/低 | ... |

## 问题与待定

- [ ] 待澄清问题 1
```
